# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report security concerns privately to:

- Email: privacy@uhorizon.ai

Include:

- A clear description of the issue.
- Steps to reproduce.
- Potential impact.
- Any suggested mitigation.

## Response Expectations

- Initial acknowledgment target: 3 business days.
- Triage and severity assessment: as soon as possible.
- Fix timeline: based on severity and complexity.

## Scope

This policy applies to framework-level content in this repository.
